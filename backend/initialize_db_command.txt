## Initialize the complete database with async batch processing

1. In terminal: [source myenv/bin/activate] to activate virtual env
2. [cd backend] to navigate to backend folder
3. [python main.py] to start the backend server
4. Use curl to initialize database:

## Default (6 stocks per batch):
curl -X POST "http://localhost:8000/api/initialize-database" \
     -H "Content-Type: application/json" \
     -d '{
       "alpha_vantage_api_key": "RMHG7PHKL60I5W5V",
       "max_stocks": 12
     }'

## Custom batch size (if your API limit is different):
curl -X POST "http://localhost:8000/api/initialize-database" \
     -H "Content-Type: application/json" \
     -d '{
       "alpha_vantage_api_key": "RMHG7PHKL60I5W5V",
       "max_stocks": 12,
       "batch_size": 6
     }'

## Notes:
- batch_size: Number of stocks processed concurrently per minute
- Default batch_size = 6 (matches Premium API limit)
- 12 stocks with batch_size=6 will take ~2 minutes (2 batches)
